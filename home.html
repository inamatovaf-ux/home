Problem 1
4x² - 8x = 0
Easy
Method (factoring):
  1. Factor out the common term: 4x(x - 2) = 0.
  2. Set each factor to zero: 4x = 0 or x - 2 = 0.
  3. Solve: x = 0 or x = 2.
Answer: x = 0, 2
Problem 2
x² - 25 = 0
Use different method
Method (square root):
  1. Move constant to the right: x² = 25.
  2. Take square root of both sides: x = ±√25.
  3. Simplify: x = ±5.
Answer: x = -5, 5
Problem 3
6x² = 0
One root
Method:
  1. Divide both sides by 6: x² = 0.
  2. Take square root: x = 0.
Answer: x = 0
Problem 4
x² + 16 = 0
No real roots
Quick check:
  1. Move constant: x² = -16.
  2. Since the right-hand side is negative, there are no real numbers whose square is negative.
  3. Therefore, no real solution.
Answer: No real solutions (нет действительных корней)
